# crud-mvc

Simple PHP CRUD using MVC architecture.


# Technologies used
**Vanilla PHP
**MySQL
**CSS
**HTML
**jQuery
**JavaScript



# Installation (Explained for windows, the steps might differ on Linux and OS X)
* Install and run XAMPP 
* Start up Apache
* Start up MySQL Server

*Download and unzip soucefiles 
* Copy crud-mvc folder to DocumentRoot "C:/xampp/htdocs/"</code> for XAMPP on Windows.
(Your folder url should be C:/xampp/htdocs/crud-mvc  

*On Xampp control panel, click on admin to go to MariaDB Server home page
*Import the file found on C:\xampp\htdocs\crud-mvc\assets\crud.example.sql
*Check your db credentials on User Accounts
*Edit the file found in C:\xampp\htdocs\crud-mvc\app\config.php and use your credentials

*Go to your prefered browser.
*Type the url :  http://localhost/crud-mvc  (Hopefully the port has not changed)
*You should then be able to access the site.
